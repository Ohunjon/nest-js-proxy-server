## Prerequisites#
Please make sure that Node.js (version >= 16) is installed on your operating system
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# you can see result in (if you have adblocker please turn of it temporary):
http://localhost:3000/

```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
